# Formul-rio-de-Cadastro-simples
Este é um projeto de estudo simples que implementa uma página de formulário de cadastro estático.

Objetivo:
O objetivo deste projeto foi praticar e demonstrar o uso de tags semânticas de formulário em HTML e a estilização e layout com CSS, separando a estrutura (`index.html`) do estilo (`style.css`).

Tecnologias Utilizadas:
HTML
CSS

Como Visualizar:
Para visualizar o projeto, basta clonar este repositório e abrir o arquivo `index.html` em qualquer navegador web.
